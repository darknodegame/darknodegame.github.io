# Dark Node: Initiation — promo site

Static promo site for **Dark Node: Initiation**, a line-of-sight puzzle for
Android. Plain HTML/CSS, served via GitHub Pages.

- `index.html` — the single-page site
- `style.css` — styling (palette + fonts mirror the game)
- `assets/img/` — branding + in-game screenshots
- `assets/fonts/` — Orbitron + Inter (SIL OFL)

No build step. Open `index.html` locally, or it's served from the repo root by
GitHub Pages.
